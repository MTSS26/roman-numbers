# Changelog

## v1.0.0

Prima release stabile del progetto.

### Funzionalità

- Conversione di numeri interi arabi da 1 a 1000 in numeri romani.
- Supporto alla notazione romana sottrattiva.
- Stampa dei numeri romani in ASCII art.

### Qualità e verifica

- Unit test automatici con JUnit.
- Analisi statica con Checkstyle.
- Controllo della coverage con JaCoCo, soglia minima 85%.
- Pipeline CI con GitHub Actions.
- Pubblicazione della coverage tramite Coveralls.
- Documentazione Javadoc dell'API pubblica.
